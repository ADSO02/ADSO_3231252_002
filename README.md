# 🚀 SIGMUI

## Smart Urban Mobility Management System

SIGMUI is a web-based platform designed to manage regulated parking zones in urban environments. The system allows citizens to reserve parking spaces, manage their vehicles, and perform payment operations, while administrators can oversee and manage the entire parking infrastructure.

---

## 📌 Project Overview

Urban parking management is a common challenge in modern cities. SIGMUI provides a centralized solution that connects citizens and administrators through a secure and scalable platform.

The system offers:

* User registration and authentication
* Vehicle management
* Parking zone monitoring
* Parking reservation management
* Payment processing simulation
* Administrative dashboard
* User and zone administration

---

## 🎯 Objective

To develop a scalable and structured software solution that optimizes urban parking management by implementing software engineering best practices and a layered architecture.

---

## 📍 Scope

SIGMUI enables users to:

### Citizens

* Register and authenticate securely
* Manage personal vehicles
* View available parking zones
* Reserve parking spaces
* Cancel reservations
* Make simulated payments
* View reservation history

### Administrators

* Manage users
* Manage parking zones
* Monitor system activity
* Access administrative dashboards
* Control platform operations

---

## 🏗️ System Architecture

SIGMUI follows a **3-Tier Architecture**:

### Presentation Layer

Responsible for the graphical user interface and user interaction.

**Technologies:**

* React

### Business Logic Layer

Handles business rules, authentication, services, and API endpoints.

**Technologies:**

* REST API
* JWT Authentication
* Node.js / Python

### Data Layer

Stores and manages application data.

**Technologies:**

* MySQL
